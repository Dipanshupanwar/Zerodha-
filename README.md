# 📈 Zerodha Clone - MERN Stack  

A fully functional **Zerodha Clone** built using the **MERN Stack**. This project consists of **two frontends** (Dashboard & Analytics) and **one backend** for handling user authentication, stock holdings, orders, and market analytics.

---

## 🚀 Features  

### 🔹 Backend (Node.js + Express.js)  
- ✅ **User Authentication** (JWT-based Login & Signup)  
- ✅ **Stock Holdings Management**  
- ✅ **Orders & Positions Tracking**  
- ✅ **MongoDB for Data Storage**  
- ✅ **REST API for Frontend Communication**  

### 🔹 Frontend 1 - Dashboard (React.js + Bootstrap)  
- 🎯 **User Login & Secure Authentication**  
- 📊 **View & Manage Stock Holdings**  
- 💰 **Buy & Sell Stocks**  
- 🌍 **Responsive UI with Bootstrap**  

### 🔹 Frontend 2 - Analytics (React.js + Chart.js)  
- 📉 **Stock Market Analysis**  
- 📈 **Interactive Graphs & Charts (Chart.js)**  
- 📋 **Real-time Orders & Holdings Data**  
- 🔍 **Investment Insights**  

---

## 🛠 Tech Stack  

| Technology  | Purpose |
|------------|---------|
| **MongoDB**  | Database for users, orders, holdings |
| **Express.js**  | Backend API framework |
| **React.js**  | Frontend framework |
| **Node.js**  | Server-side execution |
| **JWT (JSON Web Token)**  | Secure authentication |
| **Axios**  | API requests from frontend to backend |
| **Chart.js**  | Data visualization in Analytics frontend |
| **Bootstrap**  | Styling for a responsive UI |

---

## 📡 API Endpoints  

| Method | Endpoint | Description |
|--------|------------|---------------|
| `POST` | `/api/auth/signup` | User Registration |
| `POST` | `/api/auth/login` | User Login |
| `GET` | `/allHoldings` | Fetch all holdings |
| `GET` | `/allOrders` | Fetch all orders |
| `POST` | `/newOrder` | Place a new order |
